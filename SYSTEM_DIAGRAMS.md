### 1.2 Actor Descriptions

**1. Guest User (Unauthenticated)**
- Can view public pages (home, diet plan, health checker)
- Can view live statistics
- Can register or login
- Can reset forgotten password

**2. Authenticated User (Base)**
- Can edit profile and upload profile picture
- Can change password
- Can view personalized dashboard
- Can logout

**3. Donor (extends Authenticated User)**
- Can update availability status (available/unavailable)
- Can add donation records with details
- Can view complete donation history
- Can download PDF certificates
- Can earn achievement badges
- Can view active blood requests from receivers
- Can switch to receiver role
- Can contact receivers for urgent needs

**4. Receiver (extends Authenticated User)**
- Can search for available donors by blood group
- Can create urgent blood requests
- Can view all active blood requests
- Can view blood compatibility guide
- Can switch to donor role (requires blood group)
- Can contact donors directly

**5. Administrator**
- Has separate admin login portal
- Can view all registered users
- Can search users by name, email, or blood group
- Can block or unblock user accounts
- Can permanently delete users
- Can view system-wide statistics
- Can view blood group distribution charts
- Can export all user data to CSV
- Can monitor all blood requests
- Full system management capabilities
